HTML Cheatsheet

Inline elements

Tag Name Function
a A link Simply a link
img An image When you want to add an image to a page
span An inline containter Used to group elements for styling
em Emphasize text Simply emphasizes text
i Italic text Makes text italic
b Bold Makes text bold
strong Important Text Makes text heavier
mark Highlight text Highlights text
s Strikethrough text Crosses text
sub Subscript Smaller number/letter
address Address Used for addresses
cite Textual citation Used for citation
sup Superscript Small character higher than other text
q Inline quotation Quotation marks
br A line break Skips to the next line
wbr A possible line break When not spacing, that is when it skips to the next row
time Date Used to format the date
code Code format Makes text look like code
audio Audio Used for inserting audio
video Video Used for inserting audio

Block elements

Tag Name Function
p A simple paragraph Adds a paragraph to the page
blockquote An extended quotation Separates the quote from the rest of the text
details Additional information (used with -summary- and -p-) Adds details under an element
ol An ordered list Adds a numbered list
dl A definition list (dt-definition term and dd-definition of term) Defines terms
hr A horizontal rule A solid line between the lines
h1 to h6 Text Heading Headings of ddifferent sizes- 6 being the smallest
